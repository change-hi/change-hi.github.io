Module: Smart Data Collection for Sensor Networks

How to use the Tapis framework and its APIs to store and process streaming sensor data.

Prerequisite Modules

Python and Jupyter

This module provides recommended readings and introductory tutorials on Python and Jupyter.

Learning Outcomes

You understand smart data collection

  • You understand basic concepts of the Tapis Streams API and Abaco.
  • You can deploy and access a Jupyter Notebook using Mana.
  • You can use the Streams API to mimic a real-world sensor network.
